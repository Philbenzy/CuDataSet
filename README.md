

# 1 项目介绍

本缺陷数据库为铜带缺陷数据库，收集了3种不同材质的金属铜合金，分别是白铜、黄铜、紫铜约1800张，分别进行矩形框标注

# 2 拍摄参数

拍摄机器为尼康D7000，镜头为定焦50毫米，快门=1/100s，光圈=7.1，ISO=100-800，色温3000-6500K，白平衡=Auto，图片为三通道RGB图片，使用jpg格式，分辨率为800x600。尽量还原铜带自身颜色。

- Annotations：标注文件
